## 📋 タスク管理アプリ

## アプリ概要
ノジマでのスタッフ業務管理の経験から、「タスクを見える化して効率よく管理したい」という課題を感じ、自分で作成したタスク管理アプリです。

## 使用技術
- HTML
- CSS
- JavaScript

## 機能一覧
- タスクの追加・削除
- 完了チェック機能
- 優先度の設定

## 工夫した点
- JavaScriptを用いてタスクの追加・削除・完了チェックをリアルタイムで反映できるよう実装しました
- 期限・優先度を設定できる機能を加え、実務での使用を想定した設計にしました

## 苦労した点
- JavaScriptのロジック実装に苦労しましたが、エラーの原因を一つずつ調べながら解決することで、デバッグの基礎を身につけました
