## 📋 タスク管理アプリ

### 概要
前職（店舗運営・転職支援）でのタスク管理における「優先順位のあいまいさ」を解決するために開発した、実用重視の管理アプリです。

### このアプリで解決できること
* **優先順位の自動整理（JavaScript実装）**
    * 重要度（高・中・低）を選択するだけで、常に「今やるべき仕事」がリストの最上部にくるよう自動で並び替えます。
* **一瞬で判断できる視認性（CSS設計）**
    * 優先度に応じた背景色の色分けにより、忙しい現場でも文字を読み込むことなく「緊急度」を判別できます。
* **入力データの自動保存（localStorage活用）**
    * ブラウザを閉じてもデータが消えない仕組みを実装。一度入力した手間を無駄にせず、継続的な運用を可能にしています。
* **活動記録の可視化（データ取得）**
    * 追加時と完了時のタイムスタンプを自動記録。後から「いつ、どの業務に、どれだけ時間がかかったか」を振り返ることができます。

### 開発の背景と目的
* **課題**：タスクが増えてくると「付箋」や「記憶」に頼ってしまいタスク漏れが発生していた。
* **解決**：機能を「優先順位の可視化」に絞り込み、現場が直感的に使い続けられる仕組みを自力で構築した。

### 工夫した点と学び
* **技術への向き合い方**：まずは基礎を固めるため、便利なライブラリに頼らず、素のJavaScriptで全てのロジックを記述しました。
* **エラーへの対応**：動かない原因をブラウザの検証機能で特定し、一つずつ仮説と検証を繰り返して解決するデバッグの基礎を習得しました。
* **学習の実践**：ITパスポート（725点）で得た「データの保持」や「アルゴリズム」の概念をアウトプットするため、localStorageやソート機能を実装。現在は4月の基本情報技術者試験に向けて学習を継続しています。

### 今後の展望
* **分析機能の追加**：タスク完了までの所要時間を集計し、業務改善のデータとして活用する機能。
* **安全な共有機能**：現在学習中のセキュリティ知識を活かし、チームで安全に利用できる仕組みへの拡張。
